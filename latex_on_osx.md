# LaTeX on OS X

- `brew cask install basictex`

## Installing packages

```
tlmgr update --self
tlmgr install standalone
tlmgr install preview
tlmgr install ...
```

## Usage

```
/Library/TeX/texbin/pdflatex foobar.tex
```
